# 坦克大战游戏

一个使用 HTML5 Canvas 和 JavaScript 开发的经典坦克大战游戏。

## 功能特性

- 🎮 流畅的玩家控制（WASD移动，空格键射击）
- 🤖 智能敌方AI坦克
- 💥 完整的碰撞检测系统
- 🧱 随机生成的障碍物墙壁
- 📊 得分系统和生命值系统
- 🎨 现代化的游戏界面

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需要安装 http-server)
   npx http-server
   ```
   然后在浏览器中访问 `http://localhost:8000`

## 游戏操作

- **W/A/S/D** - 移动坦克（上/左/下/右）
- **空格键** - 发射子弹
- **R** - 重新开始游戏

## 游戏规则

- 玩家有 3 条生命
- 击败敌方坦克可以获得 10 分
- 被敌方子弹击中会失去 1 条生命
- 生命值归零游戏结束
- 敌方坦克会定期生成（最多同时存在 5 个）

## 技术实现

- **HTML5 Canvas** - 游戏渲染
- **JavaScript (ES6+)** - 游戏逻辑
- **CSS3** - 现代化界面设计

## 游戏类说明

- `Tank` - 坦克类，处理移动、射击和绘制
- `Bullet` - 子弹类，处理子弹移动和碰撞
- `Wall` - 墙壁类，游戏障碍物
- `Game` - 游戏主类，管理游戏状态和循环
